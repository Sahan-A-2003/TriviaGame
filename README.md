# Python Trivia Game 🎮

A simple command-line trivia game made with Python.
Test your knowledge of programming languages by answering random questions about file extensions, comment syntax, and more.

## Features

* Randomly selects 5 questions each game
* Questions about popular programming languages
* Score tracking system
* Beginner-friendly Python project
* Easy to customize with more questions

## Questions Include

* Python
* JavaScript
* Java
* C++
* HTML
* CSS
* PHP
* Ruby
* Go
* Swift
* SQL
* Shell Script
* TypeScript

## How to Run

1. Make sure Python is installed.
2. Download or clone this repository.
3. Run the file:

```bash
python trivia_game.py
```

## Example Gameplay

```text
1. What is the file extension for Python?
Your answer: .py
Correct answer..

2. What is the comment syntax for JavaScript?
Your answer: //
Correct answer..
```

```text
trivia_game.py
README.md
```

## How It Works

* Stores questions and answers in a Python dictionary
* Randomly chooses 5 questions
* Checks user answers
* Shows final score at the end
